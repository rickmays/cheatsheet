Cheatsheet

Table Tags
Tag Name Description
<table> Table The table element represents data with more than one dimension, in the form of a table.
<thead> Table Head Block of rows that consist of the column labels (headers) for the parent table
<tbody> Table Body Block of rows that consist of a body of data for the parent table
<tr> Table Row Row of cells in a table
<th> Table Header Header cell in a table
<td> Table Data Data cell in a table
<caption> Table Caption Title of the table
<tfoot> Table Footer Block of rows that consist of the column summaries (footers) for the parent table
Source: W3C Working Draft
